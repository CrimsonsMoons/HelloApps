# Aevra PC Notify

Open `AevraPCNotify.csproj` in Visual Studio 2022 with the **.NET desktop development** workload installed.

Build and run on Windows 10 2004 or later / Windows 11. Open Aevra on the iPhone and go to **Live**. Enter the pairing code once. The app discovers Aevra automatically over local Wi-Fi and falls back to Bluetooth Low Energy.

For Wi-Fi discovery, allow the Windows app through Defender Firewall on Private networks. For Bluetooth, enable Bluetooth on both devices and allow Aevra's Bluetooth permission.
